# Gestion des cours

Un projet simple en PHP pour gérer les cours entre les professeur et les étudiants.

# Fonctionnalités
- **Admin** : Crée des comptes (professeur, étudiants).
- **Professeur** : Publier les cours.
- **Etudiants** : Télécharger les cours.
